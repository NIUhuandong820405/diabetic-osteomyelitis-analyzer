# 糖尿病颌骨坏死动态CT分析工具

本代码库为《糖尿病合并阴沟肠杆菌性颌骨坏死：多模态影像动态监测引导下的精准治疗与决策体系》论文的配套代码。

## 🚀 快速开始

### 环境要求
- **Python 3.6+** （[下载地址](https://www.python.org/downloads/)）
- **无需安装任何额外库** - 简化版开箱即用

### 使用方法

#### 方案：零基础用户
```bash
# 1. 运行简化版（无需安装，直接使用）
python main_analysis_simple.py

# 2. 运行论文病例演示
python demo_usage.py

# 3. 分析您自己的病例

python interactive_analysis.py
